### 187.重复的DNA序列

```js
DNA序列 由一系列核苷酸组成，缩写为 'A', 'C', 'G' 和 'T'.。
例如，"ACGAATTCCG" 是一个 DNA序列 。
在研究 DNA 时，识别 DNA 中的重复序列非常有用。
给定一个表示 DNA序列 的字符串 s ，返回所有在 DNA 分子中出现不止一次的 长度为 10 的序列(子字符串)。你可以按 任意顺序 返回答案。

输入：s = "AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT"
输出：["AAAAACCCCC","CCCCCAAAAA"]


1.哈希表 统计 i~10 出现的次数 
2.哈希 + 滑动窗口 + 位运算
    使用二进制来表示，如下:(在代码运算的时候，要改成整型，即 A=0,c=1,g=2,t=3)
    A =  00
    C =  01
    G =  10
    T =  11
    因为要求长度10，最多就20bit,在整型的范围内。
    2.1 二进制左移2位，空出位置,给下一个字符,扩大右窗口
         x = x << 2
    2.2 添加字符str
         x = x | 哈希[str]
    2.2 确保只要低位 10*2 个bit，缩小左窗口
         x = x & ((1 << 10 * 2) - 1)

答疑：为什么要左移2位？因为左移2位置，空出了2bit，2bit位置都是0.
     怎么添加字符？或运算，0(左移后) | 1 = 1; 0(左移后) | 0=0;也就是说不管字符bit对应的      啥都可以加入，当然不能bit位置无限增加！
     bit长度怎么限制？与运算，0(原来的位置) & 1 = 0; 1(原来的位置) & 1 =1:
     1 << 10* 2，   即  1 0000 0000 0000 0000 ,低位20位都是0
     ((1<<20*2)-1), 即  0 1111 1111 1111 1111 ，低位20位都是1，高位都是0
     配合与运算，确保 20位以内的不变，20位以上的都是0
     如此做了长度限制。不会出现bit位无限增加的情况
以下是二进制减法运算：
     0-1减不了，就去高位找1去借，借了之后自身就有了2个1.....
```

### 209.长度最小的子数组

```js
给定一个含有 n 个正整数的数组和一个正整数 target 。
找出该数组中满足其总和大于等于 target 的长度最小的 连续
子数组
 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。
 
输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。

o(n)：1.右窗口持续扩大，要满足和>=target
      2.左窗口扩大，要满足和>=target
      3.计算左右窗口之间的长度。
      代码略。
o(nlogn)
      此方法使用的前缀和+二分法。
      前缀和：
      sums[]:存储前缀和,故sums是个递增的数组
      sums[i]:代表前i个数组的前缀和，
      sums[k]-sum[i]>=target,求满足条件的最小k
      
      二分法：
      1.数据递增
      2.寻找符合条件的值
      
```

### 395.至少有k个重复字符的最长子串

```js
给你一个字符串 s 和一个整数 k ，请你找出 s 中的最长子串， 要求该子串中的每一字符出现次数都不少于 k 。返回这一子串的长度。
如果不存在这样的子字符串，则返回 0。

输入：s = "aaabb", k = 3
输出：3
解释：最长子串为 "aaa" ，其中 'a' 重复了 3 次。

窗口限定:
题目给的字符串是小写英文字母，字母在'a' - 'z'范围内
限定种类区间i(1<=i<=26),表示[l,r]区间内最多有i种。
[l,r]区间的总共有total种.
total <= t,右窗口++
toltal > t,左窗口++

窗口内部逻辑:
[l,r]区间内'字符'出现次数为 num, 若num < k,则任意包含该字符的均不合格。
1.通过窗口指针移动，使得该字符出现次数为0次,则total--;
2.通过窗口指针移动，使得该字符出现次数为多次,依然右num < k
以下是细节：less记录[l,r]区间内小于k的种数。
k>=1 && total <= t，右侧++，---->  {
                          code[右]++;
                          code[右]===1----->total++;less++;
                          code[右]===k----->less--;
                     }

k>=1 && toltal > t，左侧++，----->  {
                           code[左]--;
                           code[左]===k-1---->less++;
                           code[左]===0  ---->less--;total--;
                     }
当less=0,说明[l,r]区间内无重复，长度为r-l+1
                     
```
