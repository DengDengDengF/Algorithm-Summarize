> ## 互斥锁

```js
//引用 node_modules/hash-wasm/lib/mutex.ts
class Mutex {
	private mutex = Promise.resolve();

	lock(): PromiseLike<() => void> {
		let begin: (unlock: () => void) => void = () => {};

		this.mutex = this.mutex.then(() => new Promise(begin));

		return new Promise((res) => {
			begin = res;
		});
	}

	async dispatch<T>(fn: () => PromiseLike<T>): Promise<T> {
		const unlock = await this.lock();
		try {
			return await Promise.resolve(fn());
		} finally {
			unlock();
		}
	}
}
```

*promise语法糖如下：*

```js
new Promise(begin) 
等价于 
new Promise((resolve,reject)=>{
     begin(resolve,reject)  
})
```

*第一次执行顺序如下:*

```js
await this.lock();
this.mutex.then(注册) 
new Promise((res)=>{//同步
    begin = res
})
new Promise((resove)=>{
    begin(resolve)
})//微任务执行
unlock()
```

*await this.lock()的返回值是：*

```js
等价于`new Promise(begin)语法糖`中，隐士传入的 `resolve`。`unlock`即`resolve`执行后，释放锁，执行下一个注册的`mutex回调`.
```
