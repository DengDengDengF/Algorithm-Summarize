##     动态分片

```js
并发总量为`K`，正在执行的任务有`n`个，要求最多同时执行`N`个任务,剩下`M`个任务。（潜在条件：`0 <= n <= K <= N`）
if(K == 0){//潜在条件`n = 0`
   if(N > M){
       //`M` 个任务 '循环式' 分配 `N` 个分片
   }else{
       //`N` 个任务均分配 `1` 个分片
   }
}else{//潜在条件`K > 0`,同时 `n <= N`
  if(N > M){
     //基于之前 `n` 个任务基础上，
     //`M-n` 个任务 '循环式' 分配分片
  }else{
     //基于之前 `n` 个任务基础上，
     //`N-n` 个任务 '循环式' 分配分片
  }
}
```
